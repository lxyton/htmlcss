DIVs

The DIV tag on its own doesn't do much

DIV tag definition

  • The DIV tag has no default styling
  • The DIV tag defines content areas
  • HTML is not funny

DIV tag usage

Assigning an id to a tag allows us to style that tag (and all its needed tags) directly through CSS

To give that id styling it must be named as a selector, along with its declarartion block, and preceded by a pound sign (hashtags symbol)

ID Usage

The id allows us to identify tags so that they may specifically be addressed within the CSS file
